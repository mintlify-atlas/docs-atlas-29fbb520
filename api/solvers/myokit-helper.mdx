## MyokitSimHelper

Myokit-based solver wrapper that matches the OpenCOR SimulationHelper interface.

Location: `~/workspace/source/src/solver_wrappers/myokit_helper.py:15`

### Constructor

```python

```

<ParamField path="cellml_path" type="str" required>
  Path to the CellML model file
</ParamField>

<ParamField path="dt" type="float" required>
  Time step for data storage
</ParamField>

<ParamField path="sim_time" type="float" required>
  Simulation duration (excluding pre-time)
</ParamField>

<ParamField path="solver_info" type="dict" default="None">
  Solver configuration. Supported keys:
  - `MaximumStep`: Mapped to `Simulation.set_max_step_size()`
  - `rtol`: Relative tolerance (default: 1e-8)
  - `atol`: Absolute tolerance (default: 1e-8)
  - `method`: Ignored (kept for compatibility with other helpers)
</ParamField>

<ParamField path="pre_time" type="float" default="0.0">
  Pre-simulation time for steady state
</ParamField>

### Special Features

#### CellML Preprocessing

Myokit helper includes automatic CellML preprocessing:

- **Model Flattening**: Uses libCellML to flatten models and resolve imports
- **Connection Resolution**: Properly handles variable connections across components  
- **Initial Value Expressions**: Preserves dependency of state initial values on constants
- **MathML Fixes**: Converts `max(x, 0)` functions to piecewise expressions for Myokit compatibility

Location: `~/workspace/source/src/solver_wrappers/myokit_helper.py:156`

### Example

```python
from solver_wrappers.myokit_helper import SimulationHelper

# Configure Myokit solver
solver_info = {
    'MaximumStep': 0.001,
    'rtol': 1e-7,
    'atol': 1e-9
}

# Create simulation helper
sim = SimulationHelper(
    cellml_path='model.cellml',
    dt=0.01,
    sim_time=10.0,
    solver_info=solver_info,
    pre_time=5.0
)

# Set parameters
sim.set_param_vals(['parameters.R_sys'], [1.2])

# Run simulation
if sim.run():
    results = sim.get_results([['time', 'P_lv', 'V_lv']])
    time = results[0][0]
    pressure = results[0][1]

sim.close_simulation()
```
