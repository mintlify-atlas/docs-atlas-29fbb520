## OpencorSimHelper

Simulation helper class for running CellML models using OpenCOR's CVODE solver.

Location: `~/workspace/source/src/solver_wrappers/opencor_helper.py:19`

### Constructor

```python
SimulationHelper(cellml_path, dt, sim_time, solver_info=None, pre_time=0.0)
```

<ParamField path="cellml_path" type="str" required>
  Path to the CellML model file
</ParamField>

<ParamField path="dt" type="float" required>
  Time step for data storage (simulation output interval)
</ParamField>

<ParamField path="sim_time" type="float" required>
  Simulation time duration (excluding pre-time)
</ParamField>

<ParamField path="solver_info" type="dict" default="None">
  CVODE solver configuration. Supported keys:
  - `MaximumNumberOfSteps`: Maximum steps allowed (default: 5000)
  - `MaximumStep`: Maximum step size (default: 0.0001)
  - `method`, `solver`, `dt_solver`: Ignored (kept for compatibility)
  
  Any other keys are passed directly to OpenCOR's CVODE solver properties.
</ParamField>

<ParamField path="pre_time" type="float" default="0.0">
  Pre-simulation time for reaching steady state (not stored in output)
</ParamField>

### Example

```python
from solver_wrappers.opencor_helper import SimulationHelper

# Configure solver
solver_info = {
    'MaximumStep': 0.001,
    'MaximumNumberOfSteps': 10000
}

# Create simulation helper
sim = SimulationHelper(
    cellml_path='model.cellml',
    dt=0.01,
    sim_time=10.0,
    solver_info=solver_info,
    pre_time=5.0
)

# Set parameters
sim.set_param_vals(['R_sys'], [1.2])

# Run simulation
if sim.run():
    # Get results
    results = sim.get_results([['time', 'P_lv', 'V_lv']])
    time = results[0][0]
    pressure = results[0][1]
    volume = results[0][2]

sim.close_simulation()
```
