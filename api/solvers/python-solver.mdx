## PythonSolverHelper

SciPy-based solver wrapper for running libCellML-generated Python models.

Location: `~/workspace/source/src/solver_wrappers/python_solver_helper.py:9`

### Constructor

```python
SimulationHelper(model_path, dt, sim_time, solver_info=None, pre_time=0.0)
```

<ParamField path="model_path" type="str" required>
  Path to the generated Python model file (`.py`)
</ParamField>

<ParamField path="dt" type="float" required>
  Time step for output storage
</ParamField>

<ParamField path="sim_time" type="float" required>
  Simulation duration (excluding pre-time)
</ParamField>

<ParamField path="solver_info" type="dict" default="None">
  SciPy solve_ivp configuration. Supported keys:
  - `method`: Integration method (e.g., 'RK45', 'RK23', 'LSODA', 'BDF', 'forward_euler')
  - `rtol`: Relative tolerance
  - `atol`: Absolute tolerance  
  - `max_step`: Maximum step size
  - `vectorized`: Whether RHS is vectorized
  - `dense_output`: Whether to compute dense output
  - `jac`: Jacobian function or matrix
</ParamField>

<ParamField path="pre_time" type="float" default="0.0">
  Pre-simulation time for reaching steady state
</ParamField>

### Example

```python
from solver_wrappers.python_solver_helper import SimulationHelper

# Configure SciPy solver
solver_info = {
    'method': 'LSODA',
    'rtol': 1e-7,
    'atol': 1e-9,
    'max_step': 0.001
}

# Create simulation helper
sim = SimulationHelper(
    model_path='generated_model.py',
    dt=0.01,
    sim_time=10.0,
    solver_info=solver_info,
    pre_time=5.0
)

# Set parameters
sim.set_param_vals(['R_sys'], [1.2])

# Run simulation
if sim.run():
    # Get results
    results = sim.get_results([['time', 'P_lv', 'V_lv']])
    time = results[0][0]
    pressure = results[0][1]
    volume = results[0][2]

sim.close_simulation()
```

### Supported Integration Methods

The following methods are available via `solver_info['method']`:

- **RK45**: Explicit Runge-Kutta 4(5) (default, adaptive)
- **RK23**: Explicit Runge-Kutta 2(3) (adaptive)
- **DOP853**: Explicit Runge-Kutta 8(5,3) (adaptive)
- **Radau**: Implicit Runge-Kutta (stiff problems)
- **BDF**: Backward differentiation formula (stiff problems)
- **LSODA**: Automatic stiffness detection and switching
- **forward_euler**: Simple forward Euler (educational, not recommended for production)
